# Viva Questions & Answers

1. Q: What does this project do?
   A: Records student marks, calculates totals/percentage, assigns grade and visualizes performance.

2. Q: Where is data stored?
   A: In data/grades.csv.

3. Q: How do you compute the grade?
   A: Based on percentage thresholds (A+, A, B, C, D, Fail).

4. Q: How to extend the project?
   A: GUI, DB, subject-wise analytics, user authentication.
