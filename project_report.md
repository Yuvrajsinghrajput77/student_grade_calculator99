## 1. Introduction
A CLI Python application to store marks, compute totals/percentages, assign grades and visualize class performance.

## 2. Features
- Add student records.
- Automatic grade calculation.
- CSV persistence (data/grades.csv).
- Performance graph saved as screenshots/performance_graph.png.

## 3. How to Run
Install requirements, then run `python src/main.py`.
